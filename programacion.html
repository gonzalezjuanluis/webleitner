¿Ques es un metodo de cadena?
un método de cadena es cualquier función que se aplica a un texto (string) para manipularlo o consultarlo.
Ejemplos de metodos de cadena
upper() lower() capitalize() titles() tripl() strip() rstrip() replaces() plit()
Mas ejemplos de metodos de cadena

¿Que son los inputs?
Un input es una entrada de datos que el usuario proporciona a un programa para que este lo procese.
¿Que es una lista?
Colección ordenada de elementos que puedes modificar (agregar, quitar, cambiar). numeros = [1, 2, 3]
¿Que es una tupla?
Colección ordenada pero inmutable (no se puede cambiar después de crearla).coordenada = (10, 20)
¿Que es un conjunto?
Colección sin orden y sin elementos duplicados.colores = {"rojo", "azul", "verde"}
¿Que es un diccionario?
Colección de pares clave–valor, donde cada clave es única.persona = {"nombre": "Ana", "edad": 25}
¿Que hace el bucle for?
El bucle for sirve para repetir instrucciones recorriendo una secuencia (lista, tupla, conjunto, diccionario, rango, cadena de texto, etc.).
dato interesante del buclee for
El for es como decir "para cada elemento en esta colección, haz esto". for variable in secuencia:
ejemplos del bucle for
recorrer una lista, frutas = ["manzana", "pera", "uva"] for fruta in frutas: print(fruta)
